javaScript基础——Object

发表于 2017-07-13 | 分类于 基础

一切皆对象
对象类似于属性的集合，每个属性是一个名/值对。

对象的创建

创建对象：对象直接量，关键字new，Object.create()

Object.create()为一个静态方法，不提供作为对象调用的方法。
其他静态函数
该有两个参数，第一个参数为一个需要的原型对象，第二个参数是对此对象属性的设置。

对象的属性

属性值
属性可以有自有的属性，也可从原型链继承属性。
属性访问
对象属性的访问通过.或[]的方式访问。
[]类数组方式传入属性名字符串，可以实现一些灵活的访问：

删除属性
通过delete运算符，只可以删除自身属性。
检测属性

枚举属性
for in可遍历对象所有可枚举属性（包括自有和继承）。

Object.keys()方法，返回一个数组（值为可枚举的自有属性的名称）

Object.getOwnPropertyNames()方法与keys()方法类似，返回数组，值不论是否可枚举
