DOM

发表于 2017-08-01

实验了dom一些方法

DOM :Document Object Model
DOM是一种典型的树型结构

节点

节点一般有三个属性：
nodeName 节点名称
nodeValue 节点值
nodeType 节点类型
对应的类型：
元素节点1 属性节点 2文本节点3 注释8 文档9

整个文档就是一个文档节点

操作节点

document对象
document.activeElement();返回焦点元素

document.createElement(); 创建了元素

document.createTextNode();创建了文本

document.getElementById(); 指定id的第一个元素
document.getElementsByClassName();//指定类名对象集合

document.getElementsByName(); 指定name值 的集合

document.getElementsByTagName();指定标签的集合

document.querySelector(); 更加灵活的选择器,返回匹配的第一个元素
document.querySelectorAll();返回一个集合
参数可以是class，id，或属性值

MDN上的一个例子

document.createAttribute();创建属性节点

DOM常用属性

element.appdendChild();//添加子节点
element.removeChild();//移除
element.replaceChild();//替换

element.insertBefore();//给子元素前添加兄弟节点

elment.firstChild 第一个子节点
elment.lastChild 最后一个子节点
elment.childNodes 子节点集合
elment.parentNode 父节点
elment.nextSibling 下一个兄弟节点
elment.previousSibling 上一个兄弟节点
不存在返回null

cloneNode()克隆

nodeValue() 获取节点属性的值
(节点包括属性节点，文本节点，元素节点不可用)
innerHTML() 获取或设置包括的所有html
textContent() 获取或设置所有的文本节点
获取时会保留空行与空格与换行符

标签属性

prototype ,js操作标签时的属性
Attribute ,文档标签的自有属性

attribute相关操作
element.attributes 获取属性集合(NamedNodeMap) 注意没有括号

element.getAttribute() 返回属性值
element.getAttributeNode() 返回一个arr对象(不建议使用arr对象)

element.setAttribute() 设置属性值

element.setAttributeNode() 设置属性节点
一般配合其他方法使用 如createAttribute
例

element.removeAttribute() 删除一个属性
element.removeAttributeNode() 删除属性并返回所删除的属性对象(难用…)
