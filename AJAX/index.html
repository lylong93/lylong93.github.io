http

http属于应用层
https = http + SSL/TLS 是http的安全版

http是无状态协议，不建立持久的连接 ，无记忆，不保留连接的数据

http请求的过程：
1 建立tcp连接
2 浏览器给服务器发送请求命令
3 浏览器给服务器发送请求头
4 服务器应答
5 服务器发送应答头
6 服务器发送数据
7 关闭tcp

请求(request):

包含
1.请求命令
2.请求地址url
3.请求头 :环境信息，身份验证
4.请求体 :请求正文

一般的请求方法：
GET 请求数据在url中,限制2000字符
POST 请求数据在请求体中,修改数据
HEAD 用于获取报头
PUT 取代内容
DELETE 删除内容

响应(response):

1.状态码
2.响应头 :服务器信息
3.响应体

一般的状态码：
1xx 信息类，表示收到，正在处理
2xx 成功
3xx 重定向 ，请求不成功，必须采取进一步动作
4xx 客户端错误
5xx 服务器错误

200 OK
成功
304 Not Modified
原来缓冲的文档还可以继续使用。
400 Bad Request
请求出现语法错误。
401 Unauthorized
请求未经授权
403 Forbidden
资源不可用。
404 Not Found
无法找到指定位置的资源
500 Internal Server Error
服务器遇到了意料不到的情况，不能完成客户的请求
504 Gateway Timeout
由作为代理或网关的服务器使用，表示不能及时地从远程服务器获得应答

AJAX

AjAx主要使用了XMLHttpRequest对象
主要的方法有：
关于请求
requset.open()
request.send()
关于响应
responseText: 获取响应数据(字符串)
requeseXML: 获取XML 形式响应数据
status和statusText：以数字和文本形式返回HTTP状态码
getAllResponseHeader(): 获取所有响应报头
getResponseHeader() 查询响应中某个字段的值

onreadystatechange属性：监听state改变状态
status属性：http状态码
reayState属性：
0:请求初始化
1: 服务器已建立连接 open调用
2请求已连接 处理响应头
3 请求处理 处理响应体
4 请求完成 完成响应
