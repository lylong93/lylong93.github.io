sublime text 使用

发表于 2017-07-31

sublime text 小巧方便，扩展性也很强
安装向对应的版本后，更加灵活的使用需要借助包管理组件
安装Package Control,打开sublime text,按 ‘ctrl + `’打开控制台，在控制台贴入命令(适用于3x版本)

稍等，按’shift + ctrl + p’ 可弹出Package Control对话框即安装成功.
在对话框中可以输入
install Package 查找需要安装的插件
list packages 查看已经安装插件列表
…

插件整理：
Emmet
提高编写速度。
SublimeCodeIntel
代码提示工具
BracketHighlighter
成对匹配增强，并修改括号等的颜色
DocBlockr
快速注释
ColorPicker
颜色获取
Can I Use
看到当前属性的浏览器支持情况。
AutoFileName
自动提示路径插件
A File Icon
文件支持图标
color Hightlighter
高亮颜色
…
其他可以安装使支持更多语言如(sass, vue ..)
