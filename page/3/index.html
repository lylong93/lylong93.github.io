网络3

发表于 2016-11-03 | 分类于 网络

http

http属于应用层
https = http + SSL/TLS 是http的安全版

http是无状态协议，不建立持久的连接 ，无记忆，不保留连接的数据

http请求的过程：
1 建立tcp连接
2 浏览器给服务器发送请求命令
3 浏览器给服务器发送请求头
4 服务器应答
5 服务器发送应答头
6 服务器发送数据
7 关闭tcp

请求(request):

包含
1.请求命令
2.请求地址url
3.请求头 :环境信息，身份验证
4.请求体 :请求正文

一般的请求方法：
GET 请求数据在url中,限制2000字符
POST 请求数据在请求体中,修改数据
HEAD 用于获取报头
PUT 取代内容
DELETE 删除内容

requestAnimationFrame

发表于 2016-10-27 | 分类于 深入js

一直使用setTimeout和setInterval写动画，而requestAnimationFrame是专门为动画设计的api。

大多数电脑显示器的刷新频率是60Hz，大概相当于每秒钟重绘60次。最平滑动画的最佳循环间隔是1000ms/60，约等于16.6ms。

requestAnimationFrame采用系统时间间隔，保持最佳绘制效率，使用动画卡顿不流畅，让各种网页动画效果能够有一个统一的刷新机制，从而节省系统资源，提高系统性能，改善视觉效果
