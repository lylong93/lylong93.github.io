javaScript基础——原型链

发表于 2017-07-18

javaScript中所有常用的内置对象其实都是构造函数，undefined和null除外，他们是特殊的对象

console.log(String); // function String() {}
console.log(Array);// function Array() {}
console.log(Function); //function Function() {}
console.log(undefined); // undefined
console.log(null); // object

每个函数都有一个prototype(显示原型)属性，指向一个对象
以Array为例：
console.log(Array.prototype);//
[constructor: function, toString: function, toLocaleString: function, join: function, pop: function…]
包括了一些定义的属性，方法，所有Array实例都可以拥有

每个对象又有一个proto(隐式原型)属性，指向一个对象
var a = new Array();
console.log(a.proto === Array.prototype);//true
隐式原型总指向它构造函数的显式原型

一个对象想调用属性或方法会沿着隐式原型链查找原型中的属性或方法，就形成了原型链，直到Object.prototype对象截止，

console.log(Object.prototype.proto) // null

每个对象又有一个constructor属性，指向构造它的构造函数，这样就形成了一种三角关系

console.log(typeof(undefined));//undefined
console.log(typeof(null)); //object

javaScript基础——类型，值和变量

发表于 2016-07-15 | 分类于 javaScript

值类型

javaScript分为两种类型的值：原始值（不可变的原始值），引用值（对象的引用）。

js中原始类型有：undefined,null,boolean(布尔),number(数字),string(字符串)
引用类型有：object(对象)，arry(数组)，function(函数)

当每定义一个原始类型时都是占据独立的内存空间,由于原始类型为简单的数据段，所以原始类型值存放入栈中。

a和b独自占据空间，改变b并不会影响a。
