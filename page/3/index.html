对象方法

发表于 2017-07-30 | 分类于 基础

hasOwnProperty()
返回一个布尔值判断对象是否有此属性(只包含自有属性)
isPrototypeOf()
返回一个布尔值判断对象是否在另一个对象原型链中
propertyIsEnumerable()
返回一个布尔值判断属性是否为可枚举的自身属性
toString()
返回一个表示对象的字符串（一般已被覆盖）
toLocaleString()
类似返回对象本地化字符串
valueOf()
一般有系统自动调用，如对象转数字
key()
返回对象的属性数组

MongoDB使用

发表于 2017-07-29 | 分类于 js相关

MongoDB 使用c++语言编写的NoSql(‘不仅仅是sql’)数据库之一，它提出非关系型数据库。最大的特点是高度扩展和灵活。其数据类似于json。

安装

进入官网下载安装包进行安装。
在计算机中一个盘根目录下新建一个如c:data/db文件作为数据库数据文件。
进入目录下使用命令行输入 mongod 显示 ok 说明数据库成功运行。
打开另一个命名行 输入mongo 即可以进入操作行。

基本命令

show dbs 查看所有数据库以及使用量，local数据库是系统本身创建。
use xxx(数据库名) 显示 switched to db xxx 选择使用哪一个数据库
show collections 显示当前数据库中的集合
db.xxx.drop() 删除xxx集合
db.dropDatabase() 删除当前数据库
其他增删改查类似其他数据库

express使用

发表于 2017-07-29 | 分类于 js相关

Expres是node.js主流的框架之一，（其公司还开发koa框架）。
简单理解是对node.js中API的又一次封装，使其开发更简单。

express生成器可以快速搭建项目骨架，目前已经从现有版本（4X版本）分离出去，需要额外的安装

目录大体分析：
app.js 是程序入口文件
bin 项目启动文件
package.json 关于依赖信息
public 存放公共文件
routes 关于逻辑处理的程序
views 视图文件存放

node.js简单配置

发表于 2017-07-29 | 分类于 js相关

Node.js安装

Node.js是使用javaScript语言写的服务端框架。
进入nodejs官网，下载最新的安装程序，按步骤安装到计算机中。
安装完成使用gitbush

可以显示出版本号说明安装成功。
官网经典的服务器创建，新建文件server.js

命令行中运行 node server.js，打开localhost：8080，可以看到返回到的数据。一个简单的node服务器成功运行。

NPM

npm是node包管理工具，随node安装入计算机。
平时开发可以通过npm安装一些依赖，进入npm查看多的包。
使用npm

安装的依赖都将自动放入到项目的node_modules 目录中
项目中生成的package.json 包含了依赖的详细信息。
