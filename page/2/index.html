MongoDB使用

发表于 2017-07-29

MongoDB 使用c++语言编写的NoSql(‘不仅仅是sql’)数据库之一，它提出非关系型数据库。最大的特点是高度扩展和灵活。其数据类似于json。

安装

进入官网下载安装包进行安装。
在计算机中一个盘根目录下新建一个如c:data/db文件作为数据库数据文件。
进入目录下使用命令行输入 mongod 显示 ok 说明数据库成功运行。
打开另一个命名行 输入mongo 即可以进入操作行。

基本命令

show dbs 查看所有数据库以及使用量，local数据库是系统本身创建。
use xxx(数据库名) 显示 switched to db xxx 选择使用哪一个数据库
show collections 显示当前数据库中的集合
db.xxx.drop() 删除xxx集合
db.dropDatabase() 删除当前数据库
其他增删改查类似其他数据库

关于nodejs 现有的理解

JavaScript语言的一大特点就是单线程，也就是说，同一个时间只能做一件事。那么，为什么JavaScript不能有多个线程呢？

JavaScript的单线程，与它的用途有关。作为浏览器脚本语言，JavaScript的主要用途是与用户互动，以及操作DOM。这决定了它只能是单线程，否则会带来很复杂的同步问题。比如，假定JavaScript同时有两个线程，一个线程再某个DOM节点上添加内容，另一个线程删除了这个节点，这时浏览器应该以哪个线程为准？

所以，为了避免复杂性，从一诞生，JavaScript就是单线程，这已经成了这门语言的核心特征，将来也不会改变。

为了利用多核CPU的计算能力，HTML5提出Web Worker标准，运行JavaScript脚本创建多个线程，但是子线程完全受主线程控制，且不得操作DOM。所以，这个新标准并没有改变JavaScript单线程的本质。

Node.js是使用javaScript语言写的服务端框架。

命令行中运行 node server.js，打开localhost：8080，可以看到返回到的数据。一个简单的node服务器成功运行。

使用npm
