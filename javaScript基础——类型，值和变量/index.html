javaScript基础——类型，值和变量

发表于 2016-07-15 | 分类于 基础

值类型

javaScript分为两种类型的值：原始值（不可变的原始值），引用值（对象的引用）。

js中原始类型有：undefined,null,boolean(布尔),number(数字),string(字符串)
引用类型有：object(对象)，arry(数组)，function(函数)

当每定义一个原始类型时都是占据独立的内存空间,由于原始类型为简单的数据段，所以原始类型值存放入栈中。

a和b独自占据空间，改变b并不会影响a。

当数据类型复杂时，需要占据大的空间，再进行复制赋值时，需要增加比较大的内存空间，所以将比较复杂的类型存放在堆中，而将地址放入栈中，通过引用而不是值来进行操作。

obj1和obj2其实指向一个对象，都可以影响堆中的对象。

undefined 和 null

undefined 为定义即为分配内存空间。
null 已经分配了内存空间，但未使用。可以定义一个对象为null，表示空。

typeof 和 instanceof 运算符

typeof一元运算符返回操作类型的一个字符串。

typeof 只能判断出原始值的具体类型，不能区分引用类型的具体类型。函数类型(function)特殊对待。

instanceof 能识别对象所属的类，返回true或false。
