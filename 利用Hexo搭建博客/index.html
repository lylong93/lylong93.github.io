利用Hexo搭建博客

发表于 2013-07-15 | 分类于 工具

记录利用github托管和hexo博客框架搭建个人博客

github 推出pages服务可以用于demo的演示或开源项目文档api托管。通过网上的一些文章，也可利用其搭建博客，搭建过程简单记录如下：

github操作

github不多介绍，点击进入官网注册和创建仓库，仓库名为xxx.github.io，其中的xxx必须与自身的github昵称相同 。然后配置公钥，使github远程托管与本机建立链接。

hexo配置

hexo是一个博客框架，能快速高效的搭建博客。

在电脑本地新建文件，存放博客项目文件，进入项目文件夹，使用gitbash（windows）命令行下载和安装hexo脚手架

初始化项目

安装成功后，打开_config.yml对项目进行设置，具体配置见hexo文档。
